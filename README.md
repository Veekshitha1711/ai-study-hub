# AI Master Study Hub

AI-powered learning assistant that converts videos into structured lessons.

## Features

• AI-generated summaries  
• Automatic quiz generation  
• Flashcard creation  
• Learning history tracking  
• YouTube video analysis  

## Tech Stack

Python  
Streamlit  
Google Gemini API  
yt-dlp
